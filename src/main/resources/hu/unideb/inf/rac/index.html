# Feladat Eredmény XQuery
1 Kérjük le az adatbázisban előforduló összes autót Megnyit "xquery doc('rentacardb')/company/cars"
2 Listázzuk ki a 10-es idval rendelkező autót (URL paraméterezhető) Megnyit "xquery doc('rentacardb')/company/cars/car[@id=" + id + "]"
3 Listázzuk ki az 100001-es telepen lévő összes értékesítőt (URL paraméterezhető) Megnyit "xquery element results { for $sp in doc('rentacardb')/company/sites/site[@id=" + id + "]/employee/salesperson/person/@idrefs return doc('rentacardb')/company/persons/person[@id=$sp] }"
